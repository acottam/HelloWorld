# Overview

As a software engineer preparing for a Master of Science in Data Science program, I am strengthening my development workflow and tooling foundations. This project establishes my development environment and demonstrates the end-to-end process of writing code, documenting it, and publishing it to GitHub.

This software is a simple Python program that prints a greeting to the terminal. It serves as a verification that my development environment (Python, Git, VS Code) is properly configured and ready for the more complex data analysis and programming projects I will complete this semester.

My purpose for creating this software is to establish a reliable, repeatable workflow for writing, documenting, and publishing code — a process I will follow for each module throughout the course.

[Software Demo Video](https://www.loom.com/share/063ecd90351b49798ae49406be98e6be)

# Development Environment

I developed this software using Visual Studio Code (v1.125.0) as my editor, with Git (v2.50.1) for version control, running on macOS.

The programming language used is Python (v3.13.5). No external libraries were required for this project.
